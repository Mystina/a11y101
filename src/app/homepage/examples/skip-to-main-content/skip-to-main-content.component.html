Add Skip to Main Content Link

A Skip to Main Content link needs to be added to the top of each page that has a long list of navigation links, sub-lists of links, corporate icons, site searches, and other elements before the main content. The idea of Skip to Main Content link is to provide a way to keyboard and screen reader users to jump down to an anchor or target at the beginning of the main content. Here's an example you can implement to your Angular Applications:

First, add the link effect style in your css files for the Skip to Main Content link:

skip-navigation a {{'{'}} 
    position: absolute; 
    left: -10000px; 
    top: auto; 
    width: 1px; 
    height: 1px; 
    overflow: hidden; 
} 
.skip-navigation a:focus {{'{'}} 
    position: static; 
    width: auto; 
    height: auto; 
}

Then in the HTML file in which you want to show the Skip to Main Content link, add:

<div class="skip-navigation">
    <a href="" onclick="window.location.hash='content'; event.preventDefault();">
    Skip to Main Content</a>
</div>

Then at the beginning the main content section or HTML file, add id="content", e.g.:

<section class="kape-container" id="content">
    </section>

or

<div class="sticky" *ngIf="tagListReady$ | async" id="content">
    </div>
