Make dropdown menu tab-able

According to WCAG 2.0 A 2.1.1, 2.1.2, and 2.4.3, all web page content should be accessible by keyboard keys only. Especially for elements that are CTA components on web pages, we need to make sure that tab-able functionality is enabled. For dropdown menus, we need to make sure that tabindex = "0" or tabindex with positive values if the elements should be focusable in sequential keyboard navigation.

For customized div sections on web pages, we usually combine tabindex and aria atributes aria-haspopup="true", aria-controls="[IDREF]", and aria-expanded="true" to identify the focusable areas, and the role, property, state attributes to satisfy the above WCAG standards:

A tab-able dropdown menu example exists in our TEST UI microsite Create Quiz Step One:

    <!-- Provide a big div container -->
    <div class="question-strand" [attr.strand-name]=strand.name>
    <!-- Dropdown menu button controls menu expandation, 
    menu items with correct aria attibutes - attr.aria-owns, attr.aria-controls, 
    attr.aria-pressed, attr.aria-expanded, attr.aria-haspopup -->
        <div class="flex-area"
                id="strand.name"
                role="button"
                attr.aria-pressed=true
                attr.aria-expanded=true
                attr.aria-haspopup="true"
                attr.aria-owns="strand.name_subtopics"
                attr.aria-controls="strand.name_subtopics"
                tabindex="0">
        </div>
    <!-- Accessible Menu with menu items with aria attributes -->
            <div *ngIf="strand.isOpen" 
            role="menu" 
            id="strand.name_subtopics" 
            attr.aria-labelledby="strand.name">
                <div role="menuitem" 
                attr.aria-pressed=true tabindex="0">
                    <span *ngIf="strand.hasSelectedAll else selectAll">
                    Deselect All</span>
                <div>
                <div role="menuitem" attr.aria-pressed=true tabindex="0"> 
                    <div>Check</div>
                </div>
          </div>
    </div>
