Dynamic content visually displayed should also be read by screen reaers.

The following message will be read by screen readers automatically when it's displayed on the page with triggering a CTA item:

<p *ngIf="isErrorState" role="alert">No questions available for your current selection.</p>

The following message will NOT be read by screen readers automatically when it's displayed ont the page with triggering a CTA item:

<p *ngIf="isErrorState">No questions available for your current selection.</p>
